# Tuga's App - Internship Technical & Creative Assessment

This project is my submission for the **code3x Internship Technical & Creative Assessment** (Software Engineering Intern - Web Technologies).

## Technologies Used

- React
- TypeScript
- Material-UI + MUI Icons
- react-hook-form + yup (form handling & validation)
- Firebase (Authentication – Google provider)
- React Router (navigation)
